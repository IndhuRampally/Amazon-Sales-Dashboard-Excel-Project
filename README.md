# Amazon Sales Dashboard – Excel Project
📌 Project Overview

This project showcases an interactive Amazon Sales Dashboard built using Microsoft Excel to analyze sales performance and generate meaningful business insights. The dashboard provides a comprehensive view of sales trends, product performance, customer locations, order statuses, and payment preferences through dynamic visualizations and slicers.

🎯 Objectives
Analyze overall sales performance across different product categories.
Identify top-performing products and customer locations.
Monitor order statuses to evaluate fulfillment efficiency.
Understand customer payment method preferences.
Create an interactive dashboard for data-driven decision-making.
🛠️ Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning and Transformation
Dashboard Design and Formatting

📊 Dashboard Features
Category-wise Sales Analysis – Compare sales across product categories.
Product-wise Performance – Identify products generating the highest revenue.
Customer Location Analysis – Analyze sales distribution by city.
Order Status Breakdown – Track completed, pending, and cancelled orders.
Payment Method Distribution – Understand customer payment preferences.
Interactive Slicers – Filter dashboard data dynamically by category, location, and status.

🔍 Key Insights
Electronics generated the highest sales revenue, making it the top-performing category.
Refrigerators, Laptops, and Smartphones contributed significantly to overall sales.
Miami and Denver recorded the highest sales among customer locations.
Completed orders represented the largest share of transactions, indicating efficient order processing.
PayPal and Credit Card were the most preferred payment methods.
Monitoring cancelled orders can help improve operational efficiency and customer satisfaction.
